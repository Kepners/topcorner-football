## Design System

### Color Palette
| Color | Hex | Name | Use |
|-------|-----|------|-----|
| Primary | `#1B5E20` | Pitch Green | Header, nav, primary buttons |
| Secondary | `#2E7D32` | Mid Green | Hover states, secondary UI |
| Accent | `#FFD700` | Gold | CTAs, highlights, badges |

- Body: Clean, readable
